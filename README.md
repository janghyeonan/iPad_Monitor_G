## 앱 소개

`패드를모니터로`는 USB-C iPad를 외부 입력 모니터처럼 사용할 수 있게 만든 iOS 앱입니다.  
HDMI 캡처 디바이스 또는 UVC 기반 외부 입력 장치를 iPad에 연결하면 실시간 프리뷰를 제공합니다.

### 주요 기능

- 외부 장치 자동 감지 및 실시간 프리뷰
- 화면 반전 보정 토글
- 180도 회전 토글
- 세로/가로 전환
- 화면 비율 선택(원본/채우기/4:3/16:9/21:9)
- 해상도 선택(480p/720p/1080p/4K)
- 배율 조절 및 설정 저장
- USB 디지털 오디오 출력

### 사용 조건

- USB-C 포트를 지원하는 iPad
- UVC 카메라 또는 HDMI 캡처 디바이스
- iOS 17 이상
