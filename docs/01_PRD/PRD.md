# PRD - 제품 요구사항 문서

## 1. 제품 개요
- 제품명: 패드를모니터 (`iPad_Monitor_G`)
- 목적: USB-C iPad를 캡처 입력 전용 모니터로 사용
- 핵심 문제:
  - 외부 장치(닌텐도 스위치/카메라)를 휴대 환경에서 모니터링하기 어려움
  - 거울 반전/상하 회전 문제로 즉시 사용성이 떨어짐
- 핵심 가치:
  - 연결 즉시 미리보기
  - 반전/180도/비율/배율 즉시 제어
  - USB 디지털 오디오 출력

## 2. 사용자 정의
| 사용자군 | 니즈 | 주요 사용 장면 |
|---|---|---|
| 콘솔 사용자 | 휴대 모니터 대체 | 스위치 영상 확인 |
| 카메라 사용자 | 외부 입력 모니터링 | 촬영 현장 프리뷰 |
| 테스트/개발 사용자 | 캡처 신호 확인 | 장치 검증/디버깅 |

## 3. 목표 / 비목표
### 목표 (v1)
1. 외부 UVC/캡처 장치 자동 감지
2. 안정적인 실시간 프리뷰
3. USB 디지털 오디오 재생
4. 화면 보정 컨트롤 제공

### 비목표 (v1)
1. iPhone 외부 카메라 입력 지원
2. 계정/로그인/클라우드 동기화
3. 서버 중심 스트리밍 운영 콘솔

- 성능: 선택 프리셋 기준 프리뷰 안정 동작
