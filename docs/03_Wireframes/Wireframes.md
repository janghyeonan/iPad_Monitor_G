# 와이어프레임

- 회사명: 모모스테이지 엔터테이먼트
- 담당부서: 서비스개발팀
- 담당자: 안장현
- 현재 문서 버전: v1

## 1. IA (정보 구조)
- Root
  - 프리뷰 영역
  - 상태 오버레이(미연결)
  - 하단 컨트롤 바

## 2. 로우파이 구조
```text
[전체 화면 프리뷰]
  ├─ (조건) 장치 미연결 안내
  └─ 하단 컨트롤 바
      ├─ 비율/해상도 칩 (한 줄)
      ├─ 배율 슬라이더 (+/-/리셋)
      └─ 토글 버튼(방향/오디오/반전/180도/UI)
```

## 3. 실무 기준 와이어 규칙
- 상단/중앙은 영상 표시를 우선한다.
- 설정은 하단 단일 바에 집중시켜 학습 비용을 줄인다.
- 위험한 변경(반전/회전)은 즉시 시각 피드백이 보이도록 한다.
