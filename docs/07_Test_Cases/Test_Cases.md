- 담당자: 안장현

## 1. 테스트 범위
- 외부 장치 탐지
- 프리뷰 안정성
- 오디오 재생
- 반전/회전/방향/비율/배율 UI 동작

## 3. 회귀 테스트
1. 반복 연결/해제 20회 무크래시
2. 반전/180도 연속 토글 50회 무크래시
3. 재연결 후 오디오 재생 정상
4. 해상도 전환 반복 시 프리뷰 멈춤 없음

## 5. 결함 우선순위 기준
- Critical: 앱 크래시/입력 완전 불가
- High: 오디오/보정 기능 오동작
- Medium: UI 상태 불일치
- Low: 라벨/표기 이슈
